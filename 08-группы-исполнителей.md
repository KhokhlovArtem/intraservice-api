# Группы исполнителей

Ресурс: `/api/taskexecutorgroup`

Группы исполнителей позволяют объединять пользователей в команды для назначения на заявки. Это упрощает управление исполнителями, особенно когда на сервисе работает много сотрудников.

## Метод GET. Получение групп исполнителей для сервиса

### Поля группы исполнителей для получения

| Название | Тип | Используется в search | Описание |
| :--- | :--- | :---: | :--- |
| `Id` | Int | | Идентификатор группы исполнителей |
| `Name` | String | + | Название группы исполнителей |
| `Description` | String | | Описание группы исполнителей |
| `ExecutorIds` | String | | Список идентификаторов исполнителей через запятую |
| `Executors` | String | | Список имен исполнителей через запятую |

### Получение списка групп исполнителей, назначенных на сервис

**Вызов:**
```
GET /api/taskexecutorgroup?serviceid={serviceid}&fields={fieldList}&search={searchString}&pagesize={value}&page={value}
```

#### Возможные параметры

| Параметр | Обязательность | Описание |
| :--- | :---: | :--- |
| `serviceid` | **Да** | Идентификатор сервиса. Если не указан – вернется пустой список |
| `fields` | Нет | Перечень полей ресурса «Группа исполнителей» для вывода |
| `search` | Нет | Строка поиска. Ищет в поле «Название группы исполнителей» |
| `page`, `pagesize` | Нет | Параметры отображения таблицы |

#### Примеры

**Пример 1: Вывести названия групп со списком исполнителей**
```
GET /api/taskexecutorgroup?fields=Name,ExecutorIds
```

**Пример 2: Поиск групп по подстроке "Test"**
```
GET /api/taskexecutorgroup?search=Test
```

**Пример 3: Поиск с ограничением полей**
```
GET /api/taskexecutorgroup?fields=Name,Executors&search=Тестовая
```
