# Документация API IntraService

Версия API: 5.42

Данный репозиторий содержит документацию по REST API системы IntraService. Документация переведена в формат Markdown для удобства чтения и интеграции.

## Оглавление

1.  [Общая спецификация](./00-общая-спецификация.md) - Введение, авторизация, форматы запросов и ответов, пагинация, фильтрация, мультиязычность и другие общие принципы.
2.  [Заявки](./01-заявки.md) - Получение, создание, изменение заявок, а также работа с файлами и согласованием.
3.  [Сервисы](./02-сервисы.md) - Получение списка сервисов, назначение пользователей и компаний на сервис.
4.  [Фильтры](./03-фильтры.md) - Получение списка фильтров для различных ресурсов.
5.  [Статусы](./04-статусы.md) - Получение списка статусов заявок и их фильтрация.
6.  [Приоритеты](./05-приоритеты.md) - Получение списка приоритетов заявок.
7.  [Компании](./06-компании.md) - CRUD операции с компаниями, а также работа с документами компаний и их файлами.
8.  [Пользователи](./07-пользователи.md) - CRUD операции с пользователями, получение информации о текущем пользователе.
9.  [Группы исполнителей](./08-группы-исполнителей.md) - Получение списка групп исполнителей, назначенных на сервис.
10. [Типы заявок](./09-типы-заявок.md) - Получение списка и конкретного типа заявки, информация о дополнительных полях.
11. [Жизненный цикл](./10-жизненный-цикл.md) - Получение событий жизненного цикла заявки.
12. [Трудозатраты](./11-трудозатраты.md) - CRUD операции с трудозатратами, получение доступных исполнителей.
13. [Категории](./12-категории.md) - Получение списка и конкретной категории.
14. [Активы](./13-активы.md) - CRUD операции с активами и получение типов активов.
15. [Настройки](./14-настройки.md) - Получение общих настроек системы.
16. [Токены и push-уведомления](./15-токены-и-push.md) - Добавление и удаление токенов устройств, отправка тестовых push-уведомлений.
