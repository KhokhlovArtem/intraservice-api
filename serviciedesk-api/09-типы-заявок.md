# Типы заявок

Ресурс: `/api/tasktype`

Типы заявок определяют структуру и поведение заявок в рамках сервиса. Они включают набор дополнительных полей, бизнес-настроек и правил, которые применяются при создании и работе с заявками.

## Метод GET. Получение списка типов заявки и определенного типа заявки

### Поля типа заявки для получения

| Название | Тип | Описание |
| :--- | :--- | :--- |
| `Id` | Int | Идентификатор типа заявки |
| `Name` | String | Название типа заявки |
| `Description` | String | Описание типа заявки |
| `IsArchive` | Bool | Архивный |
| `Image16Url` | String | URL картинки |
| `Changed` | DateTime | Дата изменения типа заявки |
| `CanCreateTask` | Bool | Можно ли в этом типе текущему пользователю создавать заявки. Выводится только при указании `serviceid > 0` |
| `TaskTypeFields` | TaskTypeFieldView Array | Список дополнительных полей типа заявки (только при получении определенного типа заявки) |

### Получение списка типов заявки

**Вызов:**
```
GET /api/tasktype?fields={fieldList}&serviceid={value}&archive={value}&pagesize={value}&page={value}
```

#### Возможные параметры

| Параметр | Описание |
| :--- | :--- |
| `serviceid` | Идентификатор сервиса. Позволяет получить типы заявок для определенного сервиса. Если не указан – выводятся все типы |
| `fields` | Перечень полей ресурса «Тип заявки» для вывода |
| `archive` | Фильтр по признаку «Архивный». `false` (по умолчанию) – только неархивные; `true` – все |
| `page`, `pagesize` | Параметры отображения таблицы |

#### Примеры

**Пример 1: Типы заявок для сервиса 43**
```
GET /api/tasktype?serviceid=43
```

**Пример 2: Только идентификатор и название**
```
GET /api/tasktype?fields=Id,Name
```

**Пример 3: Все типы (архивные и неархивные)**
```
GET /api/tasktype?fields=Id,Name&archive=true&pagesize=2
```

#### Пример вывода

```
GET /api/tasktype?fields=Id,Name&archive=true&pagesize=2
```

```xml
<TaskTypeList>
  <TaskTypes>
    <TaskType>
      <Id>1004</Id>
      <Name>Стандартный</Name>
    </TaskType>
    <TaskType>
      <Id>3</Id>
      <Name>Инцидент</Name>
    </TaskType>
  </TaskTypes>
  <Paginator>
    <Count>10</Count>
    <Page>1</Page>
    <PageCount>5</PageCount>
    <PageSize>2</PageSize>
    <CountOnPage>2</CountOnPage>
  </Paginator>
</TaskTypeList>
```

### Фильтрация списка типов заявки

| Поле фильтрации | Тип передаваемого значения | Описание фильтрации |
| :--- | :--- | :--- |
| `ChangedLessThan` | DateTime | Дата последнего изменения меньше или равна указанной дате |
| `ChangedMoreThan` | DateTime | Дата последнего изменения больше или равна указанной дате |
| `Changed` | DateTime | Дата последнего изменения равна указанному дню |
| `HasFields` | bool | Имеются ли у типа заявки дополнительные поля |

#### Примеры запросов с фильтрацией

**Пример 1: Типы заявок, измененные после 3 апреля 2016 года**
```
GET /api/tasktype?ChangedMoreThan=2016-04-03
```

**Пример 2: Типы заявок с дополнительными полями, измененные до 1 апреля 2016 года**
```
GET /api/tasktype?ChangedLessThan=2016-04-01&HasFields=true
```

### Получение конкретного типа заявки

**Вызов:**
```
GET /api/tasktype/{tasktypeid}?serviceid={value}
```

Где `{tasktypeid}` – идентификатор типа заявки.

Используется при создании или изменении заявки, чтобы получить информацию о типе заявки и правах текущего пользователя на редактирование его дополнительных полей.

#### Параметры

| Параметр | Обязательность | Описание |
| :--- | :---: | :--- |
| `serviceid` | Да | Идентификатор сервиса. Если на данном сервисе нет такого типа – вернет ошибку. Используется для определения прав пользователя на сервисе |

### Дополнительные поля типа заявки (TaskTypeFieldView)

| Название | Тип | Описание |
| :--- | :--- | :--- |
| `Id` | Int | Идентификатор поля |
| `Name` | String | Название поля |
| `FieldTypeId` | Int | Тип поля: `1` – Строка, `2` – Число, `3` – Дата, `4` – Чекбокс, `5` – Список, `6` – Файл, `7` - Текст |
| `IsRequired` | Boolean | Обязательное поле |
| `DefaultValue` | String | Значение по умолчанию |
| `Unit` | String | Единица измерения |
| `Hint` | String | Подсказка |
| `TaskTypeComboBoxes` | TaskTypeComboBoxView Array | Значения для выпадающего списка (тип `5`). Возвращаются в отсортированном виде |
| `Rights` | int | Сумма прав: `1` – чтение при изменении, `2` – редактирование при изменении, `4` – редактирование при создании |

### Поля значения дополнительного поля «Выпадающий список» (TaskTypeComboBoxView)

| Название | Тип | Описание |
| :--- | :--- | :--- |
| `Id` | Int | Идентификатор элемента |
| `Name` | String | Название элемента |
| `Path` | String | Путь |
| `ParentId` | Int | Идентификатор родительского элемента (для многоуровневых списков) |

Возвращаются в отсортированном виде.

```
GET /api/tasktype/1005?serviceid=16
```
